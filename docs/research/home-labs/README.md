# Home Lab Research

This folder contains research on existing home lab projects, technologies, and approaches.

## Research Focus Areas

1. **Project Examples** - Real-world home lab implementations
2. **Technology Stacks** - Common tools, platforms, and software
3. **Networking** - How multiple computers are connected
4. **Remote Access** - Methods for accessing home labs externally
5. **Use Cases** - What people build home labs for

## Research Status

- **Started:** 2026-01-20
- **Method:** Web research using Sonnet, curation using Opus
- **Phase:** Active research
